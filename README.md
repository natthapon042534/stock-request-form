# stock-request-form
# 📑 ระบบออกใบเบิกวัสดุ (Stock Request Slip)
เครื่องมือสำหรับสร้างใบเบิกวัสดุขนาด A5 ( landscape) เพื่อใช้งานภายในโรงงาน

### 🚀 ฟีเจอร์หลัก
- รองรับการสร้างหลายหน้า (Pagination)
- เลือกวันที่ผ่าน Calendar Dropdown
- พิมพ์ชื่อผู้รับ/ผู้อนุมัติ ลงในฟอร์มได้ทันที
- บันทึกเป็นไฟล์ PDF ขนาด A5 เป๊ะ

### 🛠 วิธีใช้งาน
1. เข้าใช้งานผ่านลิงก์ GitHub Pages
2. กรอกข้อมูลส่วนหัว (โรงงาน, วันที่, เลขที่เอกสาร)
3. พิมพ์รายการวัสดุ (หากรายการเยอะเกิน 8 แถว ให้กดปุ่ม "เพิ่มหน้ากระดาษ")
4. กดปุ่ม **"บันทึกเป็น PDF"** และเลือกปลายทางเป็น Save as PDF
